# C=2 プロジェクト

このリポジトリは、オリジナルのミニ言語 **C=2** を公開するためのプロジェクトです。

## C=2 とは？
C=2 は、数字と記号だけで構成される非常にシンプルなミニプログラミング言語（実験用）です。

```
C = 2
```
のように簡潔な構文を持ち、言語としての面白さとネタ要素の両方を楽しめます。

## ファイル一覧
- `Cequal2.py` — C=2 のインタプリタ（またはパーサー）を実装した Python ファイル
- `hello.c2` — サンプルコード

## 使い方
1. Python をインストール
2. `Cequal2.py` を実行し、`.c2` ファイルを読み込ませる
   ```sh
   py Cequal2.py hello.c2
   ```

## ライセンス
このプロジェクトは自由に使って構いません。
